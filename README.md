 Happy Anniversary Website

A small surprise website created to celebrate my parents' anniversary.
The website displays an animated background using **Three.js**, and when the **Celebrate 🎉** button is clicked, a photo appears with a heartfelt anniversary message.

This project was built as a fun and emotional way to create a digital celebration for my parents.

## 🌟 Features

* ✨ Animated particle background using **Three.js**
* 🎉 Interactive **Celebrate** button
* 📸 Photo reveal with smooth fade-in animation
* ❤️ Anniversary message displayed with the photo
* 🎨 Simple and elegant design


## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Three.js**
